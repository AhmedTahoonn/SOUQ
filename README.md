
#SOUQ App with Flutter

By using :

Clean code 

Dio package

Bloc as a state Management

Shared preference

-login with email and password

-Add, Update and delete Address

-Add to your cart 

-Add to your favourite list

-Update user name ,email,phone and profile image

-Search for products

-Change password
